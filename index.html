我是冯思远，本科就读于山东大学计算机科学与技术专业，硕士就读于中国科学院大学信息工程研究所。我关注大模型数据生成、模型评估、模型微调与后训练、Agent 工作流和 AI 应用落地。

在 UCAS 的硕士研究中，我围绕场景图数据生成任务，使用多模态大模型构建了高信息场景图数据生成与质量评估系统；通过可解析生成、语义密度增强、质量评估与高信息量迭代，构建了包含 903K 三元组、93.43% 生成有效率、达到 SOTA 水平的高信息量三元组数据集 MLLM-HSGG。

我近期实践大模型微调、Agent 与 vibe coding AI 应用生成：复现 nanochat 训练链路，完成 Qwen2.5-0.5B 本地 LoRA SFT / DPO 后训练，围绕数据构造清洗、模型回归测试、结构化行为评分与 badcase 数据迭代进行大模型微调与能力迭代；同时基于 LangGraph 构建论文自动生成 PPT 的 Agent 系统，串联 PDF 解析、内容规划、PPTX 渲染、布局检查与自动返修；并使用 Codex、Claude Code 制作网页思维导图应用和手机小游戏 Switch Rush，将想法快速落地并迭代推进为完整可交付的作品。

Work

精选项目

01 — 数据生成与质量评估
MLLM-HSGG
面向 VG150 标注稀疏、长尾覆盖不足与单标签评估局限，设计 MLLM 数据生成与质量评估系统；通过 prompt 约束、规则解析、词汇对齐、事实性过滤与四维评分筛选高质量数据。最终构建 903,223 条高信息三元组，生成有效率达 93.43%，高信息关系子集 mR@100 从 17.60% 提升至 48.33%。
MLLM Data Quality Evaluation VG150

02 — 后训练与行为评估
NanoChat SFT / Qwen LoRA-DPO
独立跑通 nanochat tokenizer、base train / eval、SFT 与对话推理链路；使用 Qwen2.5-0.5B 完成本地 LoRA SFT / DPO 后训练，覆盖 instruction-answer 数据清洗、preference pair 构造、adapter 保存加载、固定测试问题回归与结构化行为评分。训练后对比 base、public-SFT、custom-SFT 与 DPO 输出，依据 badcase 反向补数据，并保留表现更稳定的 SFT checkpoint。

03 — Agent 工作流实践
paper2ppt
构建论文 PDF 转可编辑 PPTX 的 Agent 原型，把 PDF 解析、内容规划、结构化 SlideSpec、PPTX 渲染、布局 QA、自动返修和 speaker script 生成串成可复用流程；针对页面空白、标题越界、内容过密和讲稿不一致等问题做多轮评估与修复，最终支持输出可编辑 PPTX 与逐页演讲稿 Markdown。
